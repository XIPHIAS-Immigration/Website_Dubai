# XIPHIAS Premium Report Images

Use this folder only for premium PDF/report visuals.

Folder format:

```text
public/images/report-assets/{country-slug}/
```

Recommended file names inside each country folder:

```text
cover.webp   Main country image. Used for the PDF cover and the large page-2 recommendation image.
img1.webp    Optional extra image. Landmark, skyline, business district, or premium destination visual.
img2.webp    Optional extra image. Passport, documents, advisor, route comparison, or process visual.
```

Use `.webp` where possible. `.jpg`, `.jpeg`, and `.png` also work.

Most countries only need `cover.webp`. Add `img1.webp` and `img2.webp` only when the existing website has too few usable images for that country.

Image guidance:

- Use commercial-safe images only: Pexels, Unsplash, Pixabay, or owned/licensed assets.
- Prefer clean, premium, bright images over dark stock-style photos.
- Avoid competitor branding, watermarks, flags as the whole image, and images with unreadable text.
- Good minimum size: `1600px` wide. Cover images are best at `2400x1600` or wider.
- Keep a small text file or spreadsheet separately with source URLs and license notes.

The report generator checks this folder first. If a file is missing, it falls back to existing website images.
